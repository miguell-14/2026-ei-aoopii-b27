# Garment Attribute Classification

Projeto desenvolvido no âmbito da unidade curricular de **AOOP II** (2026).

## 📌 Descrição
Este projeto tem como objetivo a classificação de atributos de peças de roupa a partir de imagens.  
O sistema identifica múltiplas características simultaneamente, como:

- Tipo de peça
- Cor
- Padrão (ex: riscas, xadrez)
- Estilo (casual, formal, etc.)
- Material (indícios visuais)

Trata-se de um problema de **classificação multi-label**, onde uma única imagem pode ter vários atributos associados.

## Objetivos
- Desenvolver um modelo de visão computacional para classificação de vestuário
- Explorar técnicas modernas como redes neuronais (ex: CNNs, ViT)
- Aplicar o sistema a cenários reais (e-commerce, catalogação, etc.)

## Tecnologias (planeadas)
- Python
- PyTorch / TensorFlow
- Modelos como ResNet ou Vision Transformer (ViT)
- Datasets como DeepFashion ou iMaterialist
